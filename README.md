# Puissance 4
#### *Le premier des deux joueurs à aligner 4 jetons à gagner*

Lancer le programme dans le shell avec la commande ./puissance4

------------------------------------

Ce projet a été réalisé par *Hugo MAESTRACCI* et *Guilian PIPART* durant notre première année en **G.Tech au Gaming Campus de Lyon**.
*****
Ce projet nous a introduit :
*  **Le langage C**
* **L'univers LINUX avec Debian**
* La programmation dans le shell
* Différents outils tels que emacs, git & GitHub, gcc

Ainsi que développé certaines de nos compétences :
* **Travail d'équipe**
* **Être organiser**
* **La répartition des tâches**
* **Respecter un cahier des charges**
